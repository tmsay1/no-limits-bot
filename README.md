# No Limits Discord Bot

A multi-feature Discord bot designed for community servers.

## Features

🎉 Giveaway System
• Start giveaways
• Automatic winner selection
• Participant tracking
• Winner announcements

💘 Ship Compatibility
• Fun compatibility calculator
• Percentage result
• Visual card display

🕵️ Anonymous Confessions
• Anonymous messages
• Thread discussions
• Reaction system

## Commands

/giveaway_start
/ship
/confess

## Setup

1. Install requirements
pip install -r requirements.txt

2. Create .env file
DISCORD_TOKEN=your_token

3. Run the bot
python main.py
